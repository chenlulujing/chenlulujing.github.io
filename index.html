##可见决定 APK 大小的主要是lib、 dex、res和assets，而通过对比，易车与竞品安装包差异主要存在于lib和res

lib差异


从图中可看出易车app中多了armeabi-v7a,该包的作用是可支持64位手机操作系统，这样提高了app的兼容性，但同时导致安装包变大。另外易车app里有两套视频播放器，也会增大lib大小。

res差异

易车app夜间模式通过两套不同的UI资源实现，同时也通过不同的UI资源适配了更广的机型，效果好，但是需要更多的资源。竞品的夜间模式通过简单的添加蒙层实现，效果一般，但是节省资源。

ARMv5，ARMv7 (从2010年起)，x86 (从2011年起)，MIPS (从2012年起)，ARMv8，MIPS64和x86_64 (从2014年起)，每一种都关联着一个相应的ABI
所有的x86/x86_64/armeabi-v7a/arm64-v8a设备都支持armeabi架构的.so文

##so加载方式

将SO文件直接放到libs下，然后代码中System.loadLibrary(“xxx”)；再public native static int xxx_xxx_xxx()；接下来就可以直接调用xxx_xxx_xxx()方法
